### Что объединяет лучших

- **Не белый, а «тёплый» белый**: #FAFAF7 / #F7F6F2 / #F9F8F4 вместо #FFFFFF. Снижает блики, добавляет «дорогой» оттенок.
- **Один акцентный цвет**: либо муarid violet, либо электрик-синий, либо тёплый янтарь — но **только один**. Никаких «радуг».
- **Sans-serif с высокой xHeight**: Inter, Geist, General Sans, Söhne. Иногда — serif для лого/заголовков (Anthropic).
- **Огромная типографика в hero**: 64–120px на десктопе, тонкая или средняя насыщенность.
- **Subtle подложки**: dot-grid, ультра-тонкие линии 1px с opacity ~6%.
- **Тени — почти прозрачные**: `0 1px 2px rgba(0,0,0,.04)`, иногда несколько слоёв вместо одной жирной.
- **Радиусы — 12–20px** для крупных карточек, 8px для кнопок.
- **Анимации сдержанные**: fade-in на скролл, лёгкий parallax, никаких «летающих частиц».

## 2. UX-разбор существующих сервисов поиска двойников

### Игроки рынка
- **StarByFace** — нейросеть, сравнение фото со знаменитостями.
- **Media.io Celebrity Look Alike** — принимает JPG/PNG/WEBP, мгновенный анализ.
- **MyCelebrityLookalike.com** — анализирует 68 точек лица, тренирован на миллионах изображений.
- **Fotor (Nano Banana Pro)** — анализ челюсти, расстояния между глазами, формы носа.
- **StarDoppel** — форма лица, форма глаз, пропорции лица.
- **boredhumans.com** — простой UI, мгновенный результат.

### Что у них общего (best practices)

**Загрузка фото**
- Один большой dropzone в центре экрана.
- Текст-инструкция: фото фронтально, хорошее освещение, без очков/масок, лицо одно.
- Поддержка форматов JPG/PNG/WEBP, размер до 10–15 МБ.
- Чёткий call-to-action: «Загрузите фото» / «Перетащите сюда».

**Результаты**
- Топ-3–5 совпадений с процентом схожести.
- Side-by-side сравнение пользователя и совпадения.
- Раскладка по чертам лица: глаза, нос, челюсть.

**Доверие**
- Явное «фото удаляются автоматически через 24 часа».
- Сообщение о том, что не используется для обучения моделей.
- Лого «Made with privacy in mind».

**Чего НЕ хватает у конкурентов (наши возможности отличиться)**
- Большинство выглядит дёшево, плотно набитые баннерами.
- Нет премиальной типографики.
- Нет умной системы планов (всё бесплатно, но с лимитами через рекламу).
- Нет идеи «временных уникальных источников» — это сильное отличие для нашего сервиса.

## 3. Тренды веб-дизайна 2026 (релевантные нашему проекту)

1. **Тёплые off-white фоны** вместо стерильного белого.
2. **Кинетическая типографика** — буквы появляются с задержкой, hover-переключения вариантов шрифта.
3. **Broken grid / asymmetric** — секции с разной высотой, пересекающимися элементами.
4. **Variable fonts** — один шрифт с плавной анимацией веса.
5. **AI-первый UX** — большая зона загрузки/промпта в центре экрана, минимум контролов вокруг.
6. **Storytelling scroll** — каждая секция = ясный «абзац» истории, а не набор виджетов.
7. **Privacy-first комуникация** — особенно для AI-сервисов с фото.

## 4. Выбранный для нашего проекта ориентир

Гибрид **Linear + Anthropic + Stripe**:
- База **Linear**: точная сетка, чистая типографика, светлый фон.
- Атмосфера **Anthropic**: тёплый off-white, более «человечный» тон.
- Pricing-блок **Stripe**: 3 строгие карточки с тонкими границами и одним выделенным планом.

### Цветовая палитра (зафиксирована)
```
Фон страницы          #FAFAF7  (тёплый off-white)
Фон карточек          #FFFFFF  (чистый белый — контраст)
Тёмный текст          #0A0A0A
Серый текст           #4D4D48
Тонкая граница        #E7E6E1
Акцент primary        #5B5BD6  (приглушённый violet, как у Linear)
Акцент hover          #4848C7
Успех / featured tag  #1B7A4F  (тёмно-зелёный)
Warning (лимитка)     #B5651D  (тёплый янтарь — для «доступно 24 часа»)
```

### Типографика
- **Inter** (через next/font) — единственный шрифт.
- Размеры: H1 — 72px, H2 — 48px, H3 — 24px, body — 16px, small — 14px.
- Tracking заголовков чуть отрицательный: `-0.02em`.
