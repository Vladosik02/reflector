# Doppelganger — сервис поиска двойников по фото

Скелет одностраничного приложения на **Next.js 14 (App Router) + TypeScript + Tailwind CSS**.
Светлая премиальная гамма, русскоязычный UI, монолитная вертикальная структура из трёх логических секций: интро/обучение, тарифы, функциональный блок с загрузкой фото.

> Этот репозиторий — **базовая структура (скелет)**. Дизайн, бэкенд распознавания, платежи, авторизация — следующие этапы.

---

## Что внутри

- `app/` — App Router, `layout.tsx` + `page.tsx`.
- `components/` — независимые секции страницы (Header, Hero, HowItWorks, Pricing, UploadSection, Footer).
- `lib/content.ts` — все тексты и тарифы вынесены сюда. Удобно править копирайтеру или переводить.
- `tailwind.config.ts` — дизайн-токены (цвета, типографика). Единая точка правды для цветов.
- `RESEARCH.md` — исследование рынка, тренды 2026, обоснование палитры.
- `PLAN.md` — план реализации и спецификация.

## Запуск

```bash
npm install
npm run dev
```

Откроется на `http://localhost:3000`.

## Скрипты

| Скрипт | Что делает |
|---|---|
| `npm run dev` | Дев-режим с hot reload |
| `npm run build` | Production-сборка |
| `npm run start` | Запуск собранной версии |
| `npm run lint` | ESLint |

## Дизайн-система

Палитра, типографика и обоснование выбора — в [`RESEARCH.md`](./RESEARCH.md).
Все токены живут в `tailwind.config.ts` под ключом `theme.extend.colors.brand`.

## Структура страницы

```
Header → Hero (видеоурок) → HowItWorks → Pricing → UploadSection → Footer
```

Все секции — `<section>` элементы внутри `<main>`. Прокрутка через якоря (`#tariffs`, `#upload`).

## Что ещё предстоит сделать (для следующего разработчика)

См. раздел **«Точки расширения»** в [`PLAN.md`](./PLAN.md).
Кратко:

1. Подменить плейсхолдер видео на реальный (YouTube embed / MUX / своё).
2. Подключить API распознавания лиц в `handleFile` (`UploadSection.tsx`).
3. Платежи на тарифы (ЮKassa / Stripe).
4. Бэкенд: хранение фото с TTL=24h, авто-удаление.
5. Логика «лимитированных источников» (бейдж с countdown).
6. Финальный дизайн / иллюстрации / иконки.

Все эти места помечены в коде комментариями `// TODO:`.
